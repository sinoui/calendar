# @sinoui/calendar

日历展示

[![npm version](https://img.shields.io/npm/v/@sinoui/calendar)](https://www.npmjs.com/package/@sinoui/calendar)
[![downloads](https://img.shields.io/npm/dm/@sinoui/calendar)](https://www.npmjs.com/package/@sinoui/calendar)

更多示例请参考[官方文档](https://sinoui.github.io/calendar/)。

## 依赖安装

```shell
yarn add @sinoui/calendar
```

## 入门示例

<Playground>
  {() => {
    return <Calendar />;
  }}
</Playground>

## 设置每个格子高度的日历

<Playground>
  {() => {
    return <Calendar eachHeight={100} />;
  }}
</Playground>

## 设置显示时间的日历，时分秒

<Playground>
  {() => {
    return <Calendar showTime='HH:mm:ss' />;
  }}
</Playground>

## 设置显示时间的日历,只是时分

<Playground>
  {() => {
    return <Calendar showTime='HH:mm' />;
  }}
</Playground>

## 设置只是显示时间,时分秒

<Playground>
  {() => {
    return <Calendar onlyShowTime='HH:mm:ss' />;
  }}
</Playground>

## 设置只是显示时间,时分

<Playground>
  {() => {
    return <Calendar onlyShowTime='HH:mm' />;
  }}
</Playground>

## 设置禁止选择当前日期之前的日期

<Playground>
  {() => {
    return <Calendar todayBeforeForbidden />;
  }}
</Playground>

## 设置只是选择年月

<Playground>
  {() => {
    return <Calendar onlyYearMonth />;
  }}
</Playground>

## 深色背景

import { ThemeProvider } from 'styled-components';

## API

Calendar 组件属性：

| 参数名称             | 参数类型                | 说明                       |
| -------------------- | ----------------------- | -------------------------- |
| onChange             | (value: string) => void | 日期选择事件监听           |
| eachHeight           | number                  | 每个格子的高度             |
| showTime             | string                 | 是否显示时间               |
| todayBeforeForbidden | boolean                 | 禁止选择当前日期之前的日期 |
| onlyYearMonth        | boolean                 | 只是选择年月               |
| onOk                 | ()=>void                | 点击确定按钮的回调         |
| className   | string          | 自定义类名                                                   |
| onlyShowTime   | string          | 只是显示时间    |
