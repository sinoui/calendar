# @sinoui/calendar

警告提示，展现需要关注的信息。

[![npm version](https://img.shields.io/npm/v/@sinoui/calendar)](https://www.npmjs.com/package/@sinoui/calendar)
[![downloads](https://img.shields.io/npm/dm/@sinoui/calendar)](https://www.npmjs.com/package/@sinoui/calendar)

更多示例请参考[官方文档](https://sinoui.github.io/calendar/)。

## 依赖安装

```shell
yarn add @sinoui/calendar
```

## 入门示例

<Playground>
  {() => {
    return <Calendar />;
  }}
</Playground>

## 设置每个格子高度的日历

<Playground>
  {() => {
    return <Calendar eachHeight={100} />;
  }}
</Playground>

## 设置显示时间的日历

<Playground>
  {() => {
    return <Calendar showTime />;
  }}
</Playground>

## 设置禁止选择当前日期之前的日期

<Playground>
  {() => {
    return <Calendar todayBeforeForbidden />;
  }}
</Playground>
